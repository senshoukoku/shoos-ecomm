2. Theme Toggle

Sun/moon icon button in the Navbar. Toggles dark class on <html>. Persisted in localStorage. Transition everything smoothly with transition-colors.

3. Entry Animation (Slide In)

On page load, key section elements slide in from alternating sides with staggered delays.
Uses existing slide-right keyframe + a new slide-left for alternating direction.
A usePageLoadAnimation hook manages the stagger timing.
